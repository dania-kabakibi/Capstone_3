# Capstone 3: Analysis with Power BI

## Project Overview
The goal of this project is to clean, model, analyze, and visualize business-related data using Power BI 
for EmporiUm. This project analyzes sales performance over the last four years and creates an interactive 
Power BI report to highlight key business insights, including sales trends, product performance, and 
top-selling books with their authors.

## Assigned Region
My project focuses on the **West region**, which includes:

- **Colorado**
  - Store Manager: Jim Heck

- **Online Sales**
  - Managers: Robin Gilmore and Ryann Kelly

## Analysis Included
The Power BI report includes:
- Sales trends over time
- Sales performance by category
- Sales distribution by state
- Top-selling books and authors for the region
